# Dental SaaS Frontend

## 🚀 Overview
Frontend for a Dental SaaS platform with role-based dashboards for Admin, Dentist, and Lab.

## 🧰 Tech Stack
- React.js (Vite)
- Tailwind CSS
- Axios

## 🎯 Features
- Authentication (Login/Register)
- Role-based dashboards
- Order tracking system
- Patient management
- Subscription plans UI
- Clean & responsive UI

## 📁 Folder Structure
- src/pages/
- src/components/
- src/api/

## ⚙️ Setup

```bash
git clone https://github.com/avadhutpise80/dental-saas-frontend.git
cd dental-saas-frontend
npm install
npm run dev
